## Instructions

Please submit this problem set before **11:59PM on Friday, December 12, 2025**. As usual, you will be submitting this assignment via GitHub Classroom. Your submission will be recorded when you push a commit to the repository on GitHub Classroom (instructions below), and you can make as many commits as you want before the deadline. We will grade assignments based on the latest commit.

### How to commit to GitHub

1. Open either GitHub Desktop or a command line terminal and navigate (either via the GUI or `cd` command) to the repository that contains this assignment 
2. Add the new or modified files to the staging area by either checking the box next to the files or using the command `git add <filename(s)>`
3. Commit the changes. You are required to include a commit message. This is found in the small text box near the commit button on GitHub Desktop or using the command `git commit -m 'commit message goes here'`
4. Push your changes and files to the remote repository. This is done by pressing the push to origin button on GitHub Classroom or with the command `git push`

### What to Submit

You will need to submit three deliverables for this assignment: (1) an R Markdown file, (2) a knitted file (HTML or PDF), and (3) a write-up. You can find full instructions for the project on Canvas. To generate a knitted document from RStudio, you can press the arrow next to the "Knit" button near the top panel of RStudio and select the option to knit to HTML or PDF. If you are having trouble knitting with the `{txt}` chunks, you can remove those chunks and type your answers outside of the chunks with clear labels.
